# HotKeyBind
Forge 1.16.5 command macro keybind mod.

## What it does
Bind any command or chat message to a key (with optional modifier key).

### Macro types
- **SingleUse** - fires once per press (requires key release before next)
- **Delayed** - fires after a delay when held
- **Repeating** - fires repeatedly while held
- **ToggledRepeating** - press to start/stop repeating
- **DisplayOnly** - opens chat pre-filled with the command
